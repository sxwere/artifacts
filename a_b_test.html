fixed_cell_14.pypython
from statsmodels.stats.proportion import proportions_ztest

purchase_control = df_control_clean["# of Purchase"].sum()
clicks_control   = df_control_clean["# of Website Clicks"].sum()
purchase_test    = df_test_clean["# of Purchase"].sum()
clicks_test      = df_test_clean["# of Website Clicks"].sum()

counts = np.array([purchase_test, purchase_control])
nobs   = np.array([clicks_test, clicks_control])
z_stat, p_value = proportions_ztest(counts, nobs, alternative="two-sided")

purchase_rate_control = purchase_control / clicks_control
purchase_rate_test    = purchase_test / clicks_test
lift_purchase = (purchase_rate_test - purchase_rate_control) / purchase_rate_control
✕显著差异 · 广告组更差

P 值远小于 0.001，差异具有统计显著性：测试组（投放广告）每次点击之后的购买转化率反而比对照组（PSA）低约 12%。说明点击进站之后，广告组用户的购买意愿或落地页体验存在劣势。

单次转化成本 CPA

CPA = 花费 / 购买数，逐日计算后做 Welch t 检验（不假设方差相等），比较两组日均获客成本是否有显著差异。

fixed_cell_17.pypython
import scipy.stats as stats

df_control_clean["CPA"] = df_control_clean["Spend [USD]"] / df_control_clean["# of Purchase"]
df_test_clean["CPA"]    = df_test_clean["Spend [USD]"] / df_test_clean["# of Purchase"]

cpa_A = df_control_clean["CPA"].replace([float("inf"), -float("inf")], None).dropna()
cpa_B = df_test_clean["CPA"].replace([float("inf"), -float("inf")], None).dropna()

t_stat, p_val = stats.ttest_ind(cpa_A, cpa_B, equal_var=False)
≈不显著

P 值 0.1946 远大于 0.05，未能拒绝两组单次转化成本相同的假设。尽管测试组均值高出约 17%，组内波动较大，不足以判定为真实成本差异。

from statsmodels.stats.proportion import confint_proportions_2indep, test_proportions_2indep

z_score, p_value = test_proportions_2indep(
    count1=view_counts[1], nobs1=clicks_nobs[1],
    count2=view_counts[0], nobs2=clicks_nobs[0], method="wald")

ci_low, ci_upp = confint_proportions_2indep(
    count1=view_counts[1], nobs1=clicks_nobs[1],

09

结论与建议

将六项检验结果按转化漏斗顺序汇总：广告组在漏斗两端都占优，唯独中段断崖式下跌，这正是解释"点击率涨 66%、最终购买转化率反而跌"这一看似矛盾结果的关键。

为什么点击率赢了，购买转化率却输了

把漏斗拆开看，矛盾就消失了：广告组的点击率比对照组高 66.5%，说明广告素材本身在吸引点击这件事上是成功的；用户一旦完成加购，广告组的支付完成率也比对照组高出 19 个百分点，说明结账环节、支付体验、价格本身都没有问题。问题出现在中段的两处连续断层：第一处，点击进站后能浏览到商品详情页的比例，广告组比对照组低 5.7 个百分点（30.80% vs 36.53%），说明更多广告用户在落地页就已经离开，甚至还没看到商品；第二处，浏览详情页之后是否愿意加购，广告组比对照组再低 20 个百分点（47.45% vs 66.88%），是整条漏斗里差距最大的环节。两处叠加，中段的流量质量劣势彻底抵消了两端的优势，最终拉低了整体购买转化率。

广告溢价是否值得

千次展示成本从 $21.72 涨到 $42.68，多花了近一倍的钱买展示。但综合 CPA 看，两组单次转化成本的差异（$5.05 对 $5.90）在统计上并不显著——也就是说，尽管展示更贵，最终平摊到每个成交用户身上的成本并没有被证实真的更高，CTR 的大幅提升在一定程度上抵消了 CPM 的涨幅。换句话说，钱花得更贵这件事本身不是问题，真正的问题是中段那 20 个百分点的流失：如果能把"浏览到加购"的转化率从 47% 拉回到接近对照组的水平，在点击量已经更大的基础上，最终的购买体量和单位经济性都会同步改善，而不需要先去纠结展示成本涨了多少。

行动建议

不建议简单削减广告投放或归因于广告本身无效——广告在拉点击和促成交这两端的能力都得到了数据验证。优先级最高的动作是排查广告引流落地页与广告素材承诺的一致性：用户因为广告内容点击进来，看到的商品详情页是否兑现了广告里暗示的价格、卖点或场景，错配是浏览后不加购最常见的原因。其次可以分人群、分广告创意拆解加购率，确认流失是集中在某几条素材还是全面性的，便于判断是该优化素材精准度还是该优化落地页本身。CPM 的上涨暂不需要单独处理，等中段转化修复后再评估投放规模是否需要调整。
